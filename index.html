When a browser loads a web page, it relies on doc type. It then parses the HTML and creates a document object model tree. The (DOM) tree is a hierarchical representation of the HTML elements on a web page. Each element in the DOM tree is represented as a node, and the relationships between nodes represent the structure of the HTML document

tags are the building blocks of the web page. It gives the browser infoormation on how to display content. Tags start and end with brackets. Elements are components of the web page. Start tag, content, end tag.

he cascade is the order in which the browser reads the code. Default is top to bottom, last code read takes priority and will overwrite previous code. Can be changed with the display property.
